# عرض المرفقات في الواجهة الأمامية (Modal + إطار + زر حفظ)

هذا المجلد يحتوي على ملف جاهز **للواجهة الأمامية** (بدون Bootstrap) لعرض مرفقات الخبر مع:
- زر 👁️ **مشاهدة** يفتح نافذة منبثقة (Modal)
- داخل النافذة:
 -إطار `iframe` لملفات PDF و TXT/RTF
 -عرض الصور مباشرة
 -رسالة للأنواع التي لا يدعم المتصفح عرضها
- زر ⬇️ **حفظ** لتنزيل الملف

## طريقة الاستخدام
داخل صفحة عرض الخبر في الواجهة الأمامية (حيث لديك `$pdo` و `$newsId`):

```php
require_once __DIR__ . '/frontend/news_attachments_embed.php';

gdy_render_news_attachments_embed($pdo, (int)$newsId, [
  'title' => 'مرفقات الخبر',
  // إذا موقعك داخل مجلد أو على دومين مختلف، عدّل base_url بما يناسب:
  // 'base_url' => '/',
]);
```

## ملاحظات
- PDF والصور وملفات txt غالباً يمكن عرضها داخل المتصفح.
- Word/Excel عادة لن تُعرض داخل المتصفح؛ ستظهر رسالة مع زر حفظ.
- إذا كانت الملفات على دومين مختلف أو لديك ترويسة `X-Frame-Options` تمنع التضمين، قد لا يعمل عرض iframe.

## مسار الملفات
المرفقات تُحفظ افتراضياً هنا:
- `uploads/news/attachments/`

> إذا مشروعك يخزن المرفقات في مسار مختلف، عدّل شرط السماح بالمسارات داخل `news_attachments_embed.php`.
